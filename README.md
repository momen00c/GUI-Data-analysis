# GUI-Data-analysis
Developed a Python desktop application using Tkinter and Pandas to simplify sales data analysis. The app supports importing CSV/Excel files, data cleaning, column mapping, feature engineering, KPI calculation, and exploratory data analysis (EDA). Users can preview data, remove columns, and export results to Excel.
